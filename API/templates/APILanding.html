League of Legends API Documentation

Here you will find information on consuming my api.



ADDING Champios (let riot do this first, but if you really wanted to)

POST /api/v1/champion/{name} - Add some new champions to our database

GETTING Champion Information

GET /api/v1/champion - Get all champion information
GET /api/v1/champion/<id> - Get a specific champion from our database

DELETING Champions

DELETE /api/v1/champion/<id> - Remove a champion from our database
